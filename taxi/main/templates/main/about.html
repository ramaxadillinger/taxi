Задание

Стек технологий:
Python 3.6
Redis (для хранения сессий)
PostgreSQL
Flask
Необходимо создать сайт с использованием Python 3.6 и Flask. Сайт - это сервис для диспетчерской такси.
http://127.0.0.1:8000
На страничке должна быть размещена форма для ввода номера телефона и пункта назначения.
http://127.0.0.1:8000/admin
Отображает список введенных данных с формы. Список отображается только в том случае, если пользователь авторизирован как админ.
http://127.0.0.1:8000/login
Форма для ввода данных для авторизации пользователя. После авторизации пользователя нужно сохранить в сессию.
Стили и "красивости" - на свое усмотрение
Нельзя использовать вспомогательные библиотеки для авторизации и админки.
